  1. Que es un sistema de control de versiones?

    [con el comando $ git checkout -b (nombre de la branch) --> crea y te posiciona en la branch git branch (nombre de la branch) --> solamente crea una branch ]

  4. Cómo clono un repositorio?

    [con el comando git clone y la url del repositorio]
